# Sanii Network Interoperability (SNI)

Sanii Network Interoperability (SNI) is a next-generation utility token built on OPN Chain, designed to power a decentralized creator economy, Web3 community rewards, and AI-driven gaming ecosystems.

## Project Details
- **Token Name:** Sanii Network Interoperability
- **Token Symbol:** SNI
- **Network:** OPN Chain
- **Total Supply:** 1,000,000 SNI
- **Decimals:** 18

## Contract Architecture
The repository contains `SNI.sol`, a standard smart contract compiled using Solidity `^0.8.20`. It features standard transfer mechanisms optimized for low-gas interoperable environments.
